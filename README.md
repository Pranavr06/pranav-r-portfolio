# Pranav R Portfolio

A modern full-stack portfolio and personal CMS built with Next.js, TypeScript, Supabase, and Netlify.

## Features

* Responsive modern portfolio
* Admin Dashboard with authentication
* Project, Blog, Certificate, and Testimonial management
* Draft, Publish, Archive, and Restore workflows
* Notes & Personal Knowledge Base
* Analytics & Insights Dashboard
* Media uploads using Supabase Storage
* Dark/Light theme support
* Mobile-friendly UI

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Tech Stack

* Next.js
* TypeScript
* Supabase
* PostgreSQL
* Tailwind CSS
* Netlify

## Project Structure

```txt
app/
components/
lib/
public/
supabase/
```

## Deployment

This project is deployed on Netlify and uses Supabase for backend services including authentication, database management, storage, and analytics.
